Esse trabalho é um CRUD que foi criado em PHP com MySQL, usando o XAMPP como um servidor, com a ajuda do chat gpt em algumas partes, como a parte visual e a estrutura do CRUD, pra rodar o XAMPP precisa abrir o programa e clicar na opção start no apache e no MySQL, se ficar vermelho precisa configurar o XAMPP pra isso tem que clicar em config no apache e depois em apache (httpd.conf) depois disso tem que rolar a pagina ate encontrar o listen, dai trocar o listen que estiver por 8080, mais embaixo vai ter escrito ServerName localhost: aqui tambem tem que colocar 8080 e depois disso tem que salvar o arquivo, agora tem que configurar o MySQL, clica em config e my.ini depois rolar a pagina ate encontrar escrito port: quando encontrar escrever 4406 depois dos pontos, descendo um pouco mais tem outro port: fazer a mesma coisa escrever 4406, descendo mais vai esta escrito optional - defaults to nesse tambem tem que colocar 4406 e salvar o arquivo, Pra executar o script precisa clicar em start no Apache e no MySQL depois de configurado, depois disso te que abrir o local host phpmyadmin e criar um banco de dados pra executar o script tem que clicar em importar e carregar o arquivo .sql depois clicar em executar, Para testar o CRUD precisa navegar entre as paginas do banco de dados e testar o sistema
